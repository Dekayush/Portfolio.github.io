Typing Speed Test Analyzer

The Typing Speed Test Analyzer is a web app built using HTML, CSS, and JavaScript to measure typing speed and accuracy. It displays a random paragraph, tracks input in real-time, and calculates Words Per Minute (WPM) and accuracy. The app uses timers, string comparison, and DOM manipulation to give instant feedback. It helps users improve their typing skills in an interactive way.

Video Gesture Mouse Controller

The Video Gesture Mouse Control project uses Python with OpenCV and MediaPipe to control the mouse pointer using hand gestures captured via webcam. It detects hand landmarks in real-time and maps finger movements to cursor actions like click, drag, and move. This touchless system offers an innovative way to interact with computers and demonstrates practical applications of computer vision and gesture recognition.

The Snake Game made using Python is a console or GUI-based game where the player navigates a snake to eat food and grow longer.
